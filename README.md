# KLY App

Plataforma interna para gestión de proyectos de real estate (flipping).

## Funcionalidades

* Gestión de proyectos
* Cálculo de ROI y TIR
* Seguimiento de gastos de obra
* Gestión de inversores

## Tecnología

* Frontend: HTML
* Backend: Supabase
* Deploy: Vercel

## Uso

Acceder desde el link de Vercel.

## Notas

Proyecto en desarrollo continuo.
